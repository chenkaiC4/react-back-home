## 功能介绍：
- React Native app返回桌面，后台运行插件。

## 适用平台：
- React Native Android

## 安装步骤：
- 1、运行 npm install react-back-home --save
- 2、运行 npm run configureReactBackHome

## 使用方法：
- 1、头文件定义：import { NativeModules } from 'react-native'
- 2、调用：NativeModules.BackHome.go();
